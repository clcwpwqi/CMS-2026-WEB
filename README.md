
# CMS-2026-WEB:

![Platform](https://img.shields.io/badge/Platform-Static_HTML-brightgreen)
![UI](https://img.shields.io/badge/UI-Modern_Glass-blueviolet)

这是 **CMS-2026** 算法的官方 Web 实现版。它是一个纯静态、零后端的本地加密工具。

---

## 核心特性

* **双重加密模式:**
    * **标准模式:** 使用内置静态密钥，适合快速转换不敏感信息。
    * **强加密模式:** 支持自定义私有密钥，提供极致安全保护。
* **现代化 UI:** * 响应式布局，完美适配 Android/iOS 手机端与 PC 端。
    * 支持 深色模式 与 浅色模式。
    * 内置 **中英文** 语言切换。
* **隐私至上:** 纯前端逻辑，无任何数据上传请求，支持断网离线使用。

## 部署与使用

### 直接运行 (推荐)
1. 下载本仓库中的 `index.html`。
2. 在浏览器中双击打开即可使用。

### 托管到 GitHub Pages
1. Fork 本仓库。
2. 在仓库设置中进入 `Settings > Pages`。
3. 选择 `main` 分支并点击保存。
